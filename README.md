## pythonista3_iPhone_storage_sever_build
iPhoneアプリのpythonista3を活用してデータサーバーを構築する手順です

## 市販アプリの導入
iPhoneにpythonista3をインストールしてください。
https://apps.apple.com/jp/app/pythonista-3/id1085978097

## 開発の背景 / Motivation
業務環境において、クラウドストレージ（Google Drive, Box, Dropbox等）へのアクセスが制限されている場合があります。このような制約下でも、
一時的にファイルをデバイス間で安全かつ迅速に移動・共有する手段を確保することを目的として開発しました。
本プロジェクトは、特定のプラットフォームに依存せず、
ブラウザとネットワークさえあれば「いつでも、どこでも、誰でも」データの受け渡しができる「ポータブルなデータ転送環境」の構築を目指しています。

## 特徴
- **環境を選ばない**: ブラウザ（Chrome, Edge, Safari等）があれば動作し、専用ソフトのインストールは不要。
- **双方向転送**: PCからiPhoneへのアップロード、iPhoneからPCへのダウンロードの両方に対応。
- **動的アドレス発行**: 起動時に接続用URLを自動表示し、ネットワーク環境が変わっても即座に対応可能。
- **軽量動作**: Pythonista 3標準のFlaskライブラリを利用したシンプルな設計。

## 使用方法
1. スクリプトの配置
Pythonista 3を開き、新しいスクリプトファイル（iPhone_server.py）を作成します。
本リポジトリの iPhone_server.py のコードをコピー＆ペーストして保存します。

2. サーバーの起動
スクリプトを実行（▶ボタンをタップ）します。
コンソールに以下のようなメッセージが表示されます。


サーバーを起動しました
会社PCのブラウザで以下を入力してください:
http://172.20.10.1:8080



4. 接続と操作
PC側のブラウザを開き、表示されたURL（例: http://172.20.10.1:8080）にアクセスします。
Download: リストされたファイル名をクリックするとダウンロードが始まります。
Upload: 「ファイルを選択」からファイルを選び、「Upload」ボタンを押すとiPhone内の Documents フォルダに保存されます。


## 技術スタック
Language: Python 3.x
Framework: Flask
Environment: Pythonista 3 (iOS)

## 注意事項
本ツールはローカルネットワーク内での利用を想定しています。

それぞれの環境のITポリシーに従って利用してください。

終了する際は、Pythonistaの実行画面を閉じてサーバーを停止させてください。

### ネットワークに関する補足
- **同一Wi-Fi環境**: 同一ネットワーク内であれば動作しますが、公共Wi-Fi（カフェ等）の「プライバシーセパレーター」機能により通信が遮断される場合があります。
- **テザリング推奨**: 外出先で利用する場合は、iPhoneのインターネット共有（テザリング）にPCを接続することで、確実にデータ連携を行うことができます。
